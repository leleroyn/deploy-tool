## awk bad regex 修复 (16:32)
- 问题：三个脚本（deploy.sh / backup_pj.sh / check_ports.sh）的 get_ini_value 里 `/^\\[/` 在 busybox awk 下报 `bad regex '^\\[': Missing ']'`
- 原因：awk /pattern/ 语法中 \\[ 表示字面量 \[，正确应为 /^\[/；同时配合 $0 !~ 动态变量做 section 排除
- 修复：三个脚本统一改为 `/^\[/ && $0 !~ "^\\[" section "\\]" { in_section=0 }`
